Timer for use with Trackmania United and Nations forever. 

Check https://bike.magnoren.uk/other/trackmania-tricks/ if you need help setting up TMNF/UF on Linux. 

This autosplitter requires you install the standalone version of the game from https://nadeo-download.cdn.ubi.com/trackmaniaforever/tmnationsforever_setup.exe
